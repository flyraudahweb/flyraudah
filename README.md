# Raudah Travels & Tours

Nigeria's Trusted Hajj & Umrah Partner.

## Project Overview
This is the official web application for Raudah Travels & Tours Ltd. It provides a comprehensive platform for booking Hajj and Umrah packages, managing agent relations, and handling administrative tasks.

## Key Features
- **Package Management**: Admins can create, edit, and archive Hajj/Umrah packages.
- **Booking Engine**: Customers can securely book and pay for packages via Paystack.
- **Agent Dashboard**: Dedicated portal for travel agents to manage their clients.
- **Admin Panel**: Real-time business metrics, AI-powered business assistance, and auditing.
- **Automated Communications**: Branded email notifications for bookings, payments, and security alerts.

## Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Lucide React.
- **Backend/Database**: Supabase (PostgreSQL, Edge Functions, Auth, Storage).
- **Payments**: Paystack.
- **Email**: Resend.

## Development Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file based on existing environment variables.
4. Start development server: `npm run dev`.

## Production Deployment
The application is optimized for deployment on platforms like Netlify or Vercel, with Supabase handling the backend infrastructure.
